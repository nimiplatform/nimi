# Live2D Adapter Fixture Posture

This directory intentionally contains adapter metadata only. It does not include
Live2D Cubism model binaries, textures, official sample models, or third-party
asset bytes.

The committed fixture manifest is rights-owned test metadata used to validate
Nimi Avatar adapter semantics and diagnostics. Wave 2 visual acceptance uses the
Avatar carrier deterministic harness and mocked Cubism runtime assets; operator
local third-party model packages must remain outside the repository unless
their redistribution license is explicitly admitted.
