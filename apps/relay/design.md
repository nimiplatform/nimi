# 现代 AI 聊天桌面应用 — 样式风格总结

---

## 1. 设计哲学

**核心原则：内容至上、安静界面、对话沉浸**

- 界面是对话的容器，不是主角
- 减少视觉噪音，让用户专注于思考和阅读
- 功能渐进披露（progressive disclosure）— 常用功能一键触达，高级功能按需展开

---

## 2. 色彩体系

### Dark Mode（主推）

| 用途 | Token | 色值 | 说明 |
|------|-------|------|------|
| 背景 - 最深层 | `--bg-base` | `#0A0A0A` | 主聊天区域 |
| 背景 - 侧边栏 | `--bg-surface` | `#111111` | 会话列表 |
| 背景 - 悬浮层 | `--bg-elevated` | `#1A1A1A` | 弹窗/下拉菜单 |
| 用户消息气泡 | `--bg-user-msg` | `#2A2A2A` | 微弱区分 |
| AI 消息区域 | `--bg-ai-msg` | `transparent` | 无背景，直接铺在 base 上 |
| 主强调色 | `--accent` | `#7C6FFF` | 按钮/链接/光标 |
| 文字 - 主要 | `--text-primary` | `#ECECEC` | 正文 |
| 文字 - 次要 | `--text-secondary` | `#888888` | 时间戳/辅助说明 |

| 警告 | `--warning` | `#FBBF24` | Token 用量 |

## 3. 字体系统

```
--font-sans:    "Inter", "SF Pro Display", -apple-system, sans-serif
--font-mono:    "JetBrains Mono", "Fira Code", "SF Mono", monospace
```

| 角色 | 字号 | 字重 | 行高 |
|------|------|------|------|
| 消息正文 | 15px | 400 | 1.7 |
| 代码块 | 13.5px | 400 | 1.6 |
| 内联代码 | 13.5px | 500 | inherit |
| 侧栏标题 | 13px | 500 | 1.4 |
| 侧栏条目 | 13px | 400 | 1.4 |
| 输入框 | 15px | 400 | 1.5 |
| 时间戳/标签 | 11px | 500 | 1.2 |
| 页面标题 | 20px | 600 | 1.3 |

**关键决策：**
- 消息正文 15px + 行高 1.7 = 大段文字的最佳阅读体验
- 代码块用 13.5px，比正文略小，视觉上嵌套感清晰

## 4. 间距与布局

### 全局间距 Scale

```
4px → 8px → 12px → 16px → 24px → 32px → 48px → 64px
```

### 三栏布局

```
┌──────────┬────────────────────────────────┬──────────┐
│ Sidebar  │         Chat Area              │ Detail   │
│ 260px    │    flex: 1 (max 720px居中)      │ 320px    │
│          │                                │ (可选)    │
└──────────┴────────────────────────────────┴──────────┘
```

- **侧边栏**：宽 240–280px，可折叠
- **聊天区域**：消息内容最大宽度 `720px`，居中对齐
- **右侧面板**（可选）：Artifact 预览 / 设置，宽 320–400px

### 消息间距

```
消息之间垂直间距:     24px（同一角色连续消息 8px）
消息内段落间距:       12px
消息气泡内边距:       16px 20px（用户） / 0（AI，无气泡）
头像与消息间距:       12px
```

## 5. 消息区域设计

### 用户消息

```
         ┌─────────────────────────┐
         │  用户输入的消息文字        │  ← 圆角气泡，右对齐
         └─────────────────────────┘
                              You · 2m ago
```

- 圆角：`18px`（大圆角，柔和感）
- 背景：`--bg-user-msg`
- 右对齐，最大宽度 85%

### AI 消息

```
  ◉  Model Name
  ───────────────────────────────
  AI 回复的内容，Markdown 渲染，
  支持代码块、列表、表格...

  ┌─ code ──────────────────────┐
  │ const hello = "world"       │  ← 代码块带语言标签 + 一键复制
  └─────────────────────────────┘

  👍  👎  📋  🔄               ← 操作栏，hover 时才完全显示
```

- **无气泡**，直接铺在背景上（更像文档阅读体验）
- 左侧显示模型图标 + 名称
- 底部操作栏：复制 / 重新生成 / 反馈，默认半透明，hover 显现

### 打字/流式效果

- 末尾闪烁光标：`2px 宽，20px 高，--accent 色，0.8s ease-in-out 闪烁`
- 文字逐 token 流入，无抖动（预分配空间或使用 `min-height`）
- Thinking/推理过程：折叠区域，`--text-secondary` 色，斜体

## 6. 输入区域

```
┌─────────────────────────────────────────────────────┐
│  📎  ┌─────────────────────────────────────┐   ▲   │
│      │  Message Claude...                  │   │   │
│      │                                     │  Send  │
│      └─────────────────────────────────────┘       │
│  🌐 Search   📄 Files   ⚙ Model: Claude 4         │
└─────────────────────────────────────────────────────┘
```

**规格：**
- 输入框：`auto-resize textarea`，最小高度 44px，最大高度 200px
- 圆角：`16px`（外框）/ `12px`（内输入区）
- 边框：`1px solid --border-subtle`，聚焦时 `--accent` 色描边 + 微发光
- 发送按钮：圆形 36px，`--accent` 背景，箭头图标
- 附件/工具栏在输入框下方，用小图标 + 文字标签
- 快捷键提示：`Enter 发送，Shift+Enter 换行` — 用 `--text-placeholder` 显示

## 9. 阴影与层级

```css
--shadow-sm:     0 1px 2px rgba(0,0,0,0.3);           /* 按钮 hover */
--shadow-md:     0 4px 12px rgba(0,0,0,0.4);           /* 下拉菜单 */
--shadow-lg:     0 8px 32px rgba(0,0,0,0.5);           /* 弹窗 */
--shadow-glow:   0 0 20px rgba(124,111,255,0.15);      /* 输入框聚焦 */
```

Dark mode 下阴影主要靠**亮度差**而非投影。`--bg-elevated` 比 `--bg-base` 亮一档就够了。

## 10. 动效规范

| 动作 | 时长 | 缓动 | 说明 |
|------|------|------|------|
| hover 状态 | 150ms | ease-out | 背景色/透明度变化 |
| 侧栏展开收起 | 250ms | ease-in-out | 宽度 + 透明度 |
| 消息出现 | 300ms | ease-out | opacity 0→1 + translateY 8px→0 |
| 弹窗 | 200ms | ease-out | scale 0.95→1 + opacity |
| 流式光标闪烁 | 800ms | ease-in-out | opacity 0↔1 循环 |
| 按钮点击 | 100ms | ease-in | scale 0.97 → 1 |
| 复制成功反馈 | 1500ms | — | 图标切换 + 文字 "Copied!" |

**原则：** 快速、不打断。任何动画不超过 300ms。禁用弹跳效果。

## 11. 代码块样式

```
┌─ TypeScript ─────────────────────────── 📋 Copy ─┐
│                                                    │
│  1  const greet = (name: string) => {             │
│  2    return `Hello, ${name}!`                    │
│  3  }                                              │
│                                                    │
└────────────────────────────────────────────────────┘
```

- 背景：`#161616`（比基础背景稍亮）
- 语法高亮：推荐 One Dark Pro / GitHub Dark Dimmed 色系
- 顶栏：语言标签左侧 + Copy 按钮右侧
- 行号：`--text-secondary` 色，右对齐，与代码间隔 16px
- 长代码自动折叠（超过 20 行），显示 "Show more"

## 12. Markdown 渲染规范

| 元素 | 样式 |
|------|------|
| `# H1` | 20px, 600 weight, 上方 32px 间距 |
| `## H2` | 17px, 600 weight, 上方 24px 间距 |
| `### H3` | 15px, 600 weight, 上方 20px 间距 |
| `**bold**` | 600 weight |
| `*italic*` | italic |
| `inline code` | `--font-mono`, 13px, `#2A2A2A` 背景, 4px 内边距, 4px 圆角 |
| 链接 | `--accent` 色, 下划线 hover 显示 |
| 引用块 | 左边框 `3px --accent`, 左内边距 16px, `--text-secondary` 色 |
| 分隔线 | `1px --border-subtle`, 上下 24px margin |
| 列表 | 左缩进 24px, 项间距 6px |
| 表格 | 无外边框, 行分隔线 `--border-subtle`, 交替行微着色 |

---

## 13. 图标风格

- 线性图标（stroke-based），1.5px 描边
- 推荐图标库：**Lucide** 或 **Phosphor**
- 尺寸标准：16px（内联）、20px（按钮）、24px（导航）
- 颜色跟随 `--text-secondary`，交互时切换 `--text-primary`

- 居中布局，占聊天区域垂直 40% 位置

- 输入框底部提示："Claude can make mistakes. Verify important information."

## 16. 无障碍要求

- 所有文字对比度 ≥ 4.5:1（WCAG AA）
- 键盘可达所有交互元素，Tab 顺序逻辑清晰
- 焦点指示器：`2px --accent outline, 2px offset`
- 代码块支持屏幕阅读器（`role="region"`, `aria-label`）
- 减弱动效：`prefers-reduced-motion` 时禁用所有非必要动画
