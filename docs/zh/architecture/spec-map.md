- Realm 互联精简映射：[`spec/realm/app-interconnect-model.md`](https://github.com/nimiplatform/nimi/blob/main/spec/realm/app-interconnect-model.md)
- AI Agent 安全摘要（一页）：[`docs/architecture/ai-agent-security-interface-summary.md`](./ai-agent-security-interface-summary.md)
- AI Agent 安全白皮书（对外）：[`docs/architecture/ai-agent-security-interface.md`](./ai-agent-security-interface.md)
