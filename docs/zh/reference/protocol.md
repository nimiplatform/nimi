# 协议参考

Nimi 协议契约采用分层设计，由仓库真实来源系统管理。

## 层级

- L0 核心信封（Core Envelope）
- L1 Runtime 访问层（Runtime Access）
- L2 Realm 核心配置层（Realm Core Profile）

## 使用规则

本页面作为面向开发者的说明文档。规范性协议规则请参见 `spec/`。
