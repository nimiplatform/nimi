Nimi welcomes third-party contributions across runtime, SDK, desktop, web, docs, and tooling.

## Start here

- Contribution policy: [`CONTRIBUTING.md`](https://github.com/nimiplatform/nimi/blob/main/CONTRIBUTING.md)
- Onboarding: [`ONBOARDING.md`](https://github.com/nimiplatform/nimi/blob/main/ONBOARDING.md)
- Security policy: [`SECURITY.md`](https://github.com/nimiplatform/nimi/blob/main/SECURITY.md)
- Open-source release runbook: [Open-Source Release](./open-source-release.md)

## Community

Join our [Discord](https://discord.gg/BQwHJvPn) to ask questions, discuss ideas, and get help from the community.

## Contribution expectations

- Keep changes scoped and contract-consistent.
- Update developer docs when behavior changes.
- Run relevant checks before opening a PR.
